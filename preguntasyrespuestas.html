  • Preguntas y Respuestas

1.¿Qué es una red informática?

Una red informática es un conjunto de dispositivos electrónicos interconectados que comparten recursos y servicios para facilitar la comunicación y el intercambio de información.

2.¿Qué es una red LAN?

Una red de área local (LAN) es una red que conecta dispositivos en un área geográfica limitada, como un edificio o una empresa. Los dispositivos en una LAN pueden compartir recursos y comunicarse directamente entre sí.

3.¿Qué es una red WAN?

Una red de área amplia (WAN) es una red que abarca una geografía más extensa, como una ciudad, un país o incluso a nivel mundial. Las WAN utilizan tecnologías como conexiones telefónicas, fibra óptica o satélites para conectar redes LAN distantes.

4.¿Qué es Internet?

Internet es una red global de redes interconectadas que permite la comunicación y el intercambio de información a nivel mundial. Utiliza protocolos estándar para la transferencia de datos, como el Protocolo de Control de Transmisión (TCP) y el Protocolo de Internet (IP).

5.¿Qué es la Web?

La World Wide Web (WWW) o simplemente la Web es un sistema de información en el que documentos y recursos multimedia están vinculados entre sí a través de enlaces. Los usuarios acceden a estos recursos utilizando navegadores web.

6.¿Qué es WWW?

WWW significa World Wide Web y se refiere a la colección de documentos y recursos multimedia vinculados entre sí mediante enlaces. Es una parte de Internet y se accede a través de navegadores web.

7.¿Qué es el protocolo HTTP y HTTPS?

HTTP (Hypertext Transfer Protocol) es un protocolo de comunicación que permite la transferencia de información en la Web. HTTPS (Hypertext Transfer Protocol Secure) es una versión segura de HTTP que utiliza cifrado SSL/TLS para proteger la información transmitida.

8.Página web estática vs página web dinámica.

Una página web estática muestra contenido fijo y no cambia sin intervención manual en el código. Una página web dinámica se genera en tiempo real, permitiendo interactividad y personalización según la interacción del usuario.

9.CMS. Ejemplos. Ventajas e inconvenientes.

Un CMS (Sistema de Gestión de Contenidos) es una plataforma que facilita la creación y gestión de contenido web. Ejemplos incluyen WordPress, Joomla y Drupal. Ventajas: facilidad de uso, flexibilidad. Inconvenientes: posibles problemas de seguridad, aprendizaje inicial.

10.Proveedor de servicios de Internet. Ejemplos.

Proveedores de servicios de Internet (ISP) ofrecen acceso a Internet. Ejemplos incluyen Comcast, AT&T, y Telefónica.

11.Dirección IP. Diferencia entre una dirección IP privada y pública.

Una dirección IP identifica un dispositivo en una red. Las direcciones IP privadas se utilizan dentro de una red local, mientras que las direcciones IP públicas son asignadas a dispositivos directamente accesibles desde Internet.

12.¿Qué es el alojamiento web/hosting? Ejemplos.

El alojamiento web o hosting es el servicio que permite almacenar y publicar páginas web en Internet. Ejemplos de proveedores de hosting incluyen Bluehost, HostGator y AWS (Amazon Web Services).
